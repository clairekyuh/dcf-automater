# Intrinsic — DCF Automater

A simple, transparent discounted cash flow workbench built with Next.js and TypeScript.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Current features

- Editable operating and valuation assumptions
- Five-year cash-flow projection
- Enterprise value, equity value, and implied value per share
- Bear, base, and bull growth scenarios
- Guardrail for invalid WACC and terminal-growth assumptions
- Responsive layout for desktop and mobile

This first version uses user-entered sample data. A future version can connect to a licensed financial-data API for automatic company data.
